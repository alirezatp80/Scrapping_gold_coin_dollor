# Price Scraper

یک اسکریپت پایتون برای دریافت قیمت لحظه‌ای **ارز دیجیتال، ارزهای خارجی و طلا/سکه** از وب‌سایت [TGJU](https://www.tgju.org/).

---

## قابلیت‌ها
- نمایش قیمت بیت‌کوین، اتریوم و تتر به ریال
- نمایش قیمت ارزهای مهم (دلار، یورو، پوند، دینار کویت و غیره)
- نمایش قیمت سکه و طلا
- قالب‌بندی قیمت‌ها با جداکننده هزارگان

---
